# Survive it

`Survive it` is a Godot 4 top-down fortress survival prototype inspired by Warcraft 3 custom games.

## Stack

- Engine: Godot 4
- Language: GDScript
- View: 2D top-down

## Current Scaffold

- Godot project configuration
- Main entry scene
- Basic world scene
- Movable hero prototype
- Starter wave manager
- Planning docs

## Open The Project

1. Install Godot 4.
2. Open Godot.
3. Import this folder:
   `C:\Users\Verdiner\Desktop\Survive it`
4. Run the project.

## First Development Goal

Build one playable loop:

- move the hero
- survive timed enemy waves
- place a few structures
- earn and spend resources
