# Survive it - First Milestone

## Milestone Goal

Create a playable vertical slice with one map, one controllable hero, one builder, one fortress to defend, one resource loop, and simple enemy waves.

## Core Systems

- Fortress health and loss condition
- Mouse-driven hero and builder command flow
- Hero combat
- Builder unit basics
- Enemy spawning and lane pressure
- Building placement
- Resource income and spending
- Wave progression
- Basic HUD

## Build Order

1. Fortress and lose condition
2. Enemy path to the fortress
3. Enemy wave spawner
4. Mouse-driven unit selection and command input
5. Hero combat
6. Builder and building placement grid
7. Resource counter and rewards
8. One defensive structure
9. Win/lose loop
10. Two more building types
11. Nightly phase loop

## Rules For Development

- Keep one script responsible for one clear system
- Keep gameplay values configurable
- Avoid premature multiplayer work
- Prefer simple shapes and placeholders over art polish
